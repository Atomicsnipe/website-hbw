About Hot Beans Web

Founded in 2017, Hot Beans Web is a full-service UK web development agency. We build fast, accessible, and well-designed applications for clients in e-commerce, healthcare, education, and the public sector.

We've grown from a 3-person startup to a team of 18 developers, designers, and project managers. Crucially, most of our senior staff started their career with us directly after completing qualifications — just like the ones we're recruiting for now.

We believe that a junior developer with strong fundamentals and the right attitude can be shaped into an excellent engineer. That's why we invest heavily in mentorship and training from day one.

Current vacancies

Two junior roles are open now. Both are suitable for people who have recently completed a relevant course, bootcamp, or qualification.

Junior Frontend Developer
Full-time Remote £24,000–£28,000 Engineering

Role Overview

You will work within our frontend team building responsive, accessible web interfaces for client projects. You'll be paired with a senior developer mentor and involved in live client work from week one.

Key Responsibilities

  • Build responsive UIs using HTML5, CSS3, and JavaScript (ES6+)
  • Develop reusable components in React
  • Implement designs from Figma with pixel accuracy
  • Write clean code and participate in peer code reviews
  • Carry out cross-browser and cross-device testing
  • Contribute to WCAG 2.1 accessibility audits

Required Qualifications

  • Level 3 qualification in IT, Computing, or Software Development
  • Completion of an accredited web development course or bootcamp
  • Solid understanding of HTML, CSS, and JavaScript
  • Use of Git/GitHub in a project context
  • Portfolio of at least 2 web projects
  • GCSE Maths and English (Grade C/4 or above)

Desirable (Not Essential)

  • Experience with React or another JavaScript framework
  • Familiarity with CSS preprocessors or Tailwind CSS
  • Awareness of web performance optimisation techniques
Junior Backend Developer
Full-time London / Remote £25,000–£30,000 Engineering

Role Overview

You will join our backend team to build and maintain the APIs, databases, and infrastructure that power our platforms. Real responsibility and a dedicated mentor from your first day.

Key Responsibilities

  • Build RESTful APIs using Node.js (Express) or Python (Django/Flask)
  • Design and manage relational databases (PostgreSQL / MySQL)
  • Write unit and integration tests to maintain code quality
  • Deploy and monitor applications on cloud platforms (AWS / GCP)
  • Implement secure authentication and input validation
  • Document APIs using OpenAPI / Swagger standards

Required Qualifications

  • Level 3 qualification in Computing, Software Development, or equivalent
  • Completion of a coding bootcamp or programming course
  • Proficiency in at least one server-side language (Python, Node.js, PHP)
  • Basic SQL and relational database knowledge
  • Understanding of HTTP, REST, and JSON
  • Use of Git for version control

Desirable (Not Essential)

  • Experience with Docker or containerised workflows
  • Familiarity with CI/CD pipelines (e.g. GitHub Actions)
  • Exposure to NoSQL databases (MongoDB)
