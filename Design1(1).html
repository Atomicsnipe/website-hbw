About Hot Beans Web

Who are we?

Hot Beans Web is a full-service web development company based in the UK, specialising in building fast, accessible, and beautifully designed web applications for clients across e-commerce, healthcare, education, and the public sector.

Founded in 2017, we have grown from a team of three to over 18 developers, designers, and project managers. We believe in nurturing talent — the majority of our senior team joined us straight from bootcamps or level 3 qualifications, exactly like the roles we are advertising now.

We partner with clients ranging from local SMEs to national charities, delivering websites and web apps that are performant, WCAG-compliant, and maintainable for years to come. Our work has won two regional digital industry awards.

Open Positions

Current vacancies

We are currently hiring two junior developer roles. Both are open to candidates who have recently completed a relevant qualification, bootcamp, or self-study programme.

Junior Frontend Developer
Full-time Remote (UK) £24,000 – £28,000

Join our frontend team to build responsive, accessible web interfaces for clients across the UK. You will be mentored by a senior developer and take ownership of real client-facing features from your first month.

Key Responsibilities
  • Build responsive UIs using semantic HTML5, CSS3, and JavaScript (ES6+)
  • Develop reusable React components for our shared component library
  • Implement designs from Figma files with pixel-level accuracy
  • Write clean, documented code and take part in regular peer code reviews
  • Cross-browser and device testing to ensure compatibility
  • Perform accessibility audits against WCAG 2.1 AA standards
  • Optimise pages for performance using Lighthouse and Core Web Vitals
Required Qualifications & Skills
  • Level 3 qualification in IT, Computing, or Software Development (e.g. BTEC, A-Level)
  • Accredited web development bootcamp or equivalent self-study with portfolio evidence
  • Solid understanding of HTML, CSS, and JavaScript fundamentals
  • Version control experience with Git and GitHub
  • Portfolio of at least 2 completed web projects (academic or personal)
  • GCSE Maths and English at Grade C / 4 or above
  • Clear written and verbal communication skills
  • Comfortable working in an agile sprint environment
Desirable (Not Essential)
  • Experience with React, Vue, or another JavaScript framework
  • Knowledge of CSS pre-processors (Sass) or utility frameworks (Tailwind)
  • Awareness of web performance, SEO fundamentals, and accessibility tooling
  • Familiarity with Figma or similar UI design tools
Junior Backend Developer
Full-time London / Remote £25,000 – £30,000

Join our backend team to build and maintain the APIs, databases, and server infrastructure that power our client applications. Expect real responsibility and strong mentorship from day one.

Key Responsibilities
  • Develop RESTful APIs using Node.js (Express) or Python (Django / Flask)
  • Design and manage relational databases using PostgreSQL or MySQL
  • Write unit and integration tests to maintain code reliability
  • Deploy and monitor applications on AWS or GCP
  • Implement server security including input validation and authentication (OAuth2 / JWT)
  • Document APIs using OpenAPI / Swagger specifications
  • Participate in sprint planning, standups, and retrospectives
Required Qualifications & Skills
  • Level 3 qualification in Computing, Software Development, or equivalent
  • Completion of a coding bootcamp or programming-focused course
  • Working knowledge of at least one server-side language (Python, Node.js, or PHP)
  • Basic SQL and relational database design understanding
  • Familiarity with HTTP, REST APIs, and JSON data structures
  • Experience using Git for version control in a real project context
  • Strong analytical and debugging mindset
  • GCSE Maths and English at Grade C / 4 or above
Desirable (Not Essential)
  • Experience with Docker or containerised development environments
  • Awareness of CI/CD pipelines (e.g. GitHub Actions, CircleCI)
  • Exposure to NoSQL databases such as MongoDB or Redis
  • Basic understanding of microservices architecture patterns
